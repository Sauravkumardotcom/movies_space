## Quick Deploy (No Code Changes Needed!)

### Option 1: Automatic (Recommended)

1. Go to [vercel.com](https://vercel.com) and log in
2. Click **"Add New..."** ‚Üí **"Project"**
3. Select **"Import Git Repository"**
4. Search for `Sauravkumardotcom/movies_space`
5. **Root Directory**: Select `movies_space`
6. Add these Environment Variables:
   ```
   VITE_API_BASE_URL=https://[YOUR_PROJECT].vercel.app/api
   VITE_GOOGLE_APPS_SCRIPT_URL=[YOUR_GOOGLE_SCRIPT_URL]
   VITE_ADMIN_EMAIL=[YOUR_EMAIL]
   ```
7. Click **"Deploy"** ‚Üí Done! üéâ

## Troubleshooting

**Issue**: Build fails with "Root directory not found"
- Fix: Go to Settings ‚Üí General ‚Üí Set Root Directory to `movies_space`

**Issue**: API not responding (CORS error)
- Fix: API endpoint should be `https://your-url.vercel.app/api` (not localhost)

**Issue**: Environment variables not loading
- Fix: Wait 2 minutes after adding, then redeploy (Settings ‚Üí Git ‚Üí Redeploy)

**Issue**: Pages work but styles look wrong
- Fix: Clear browser cache (Ctrl+Shift+Delete) and hard refresh (Ctrl+Shift+R)
